# نظام تبادل الزيارات التلقائي

## الوصف
تطبيق ويب لتبادل الزيارات التلقائي بين المواقع، حيث يمكن للمستخدمين إضافة روابط مواقعهم والحصول على زيارات من مواقع أخرى مشاركة في النظام.

## الميزات الرئيسية
- إضافة روابط المواقع إلى قائمة التبادل
- تبادل زيارات تلقائي كل 10 ثواني
- عرض إحصائيات الزيارات (المواقع المزارة وعدد الزوار)
- قائمة بجميع الزوار والمواقع المضافة
- عداد تنازلي للانتقال التلقائي
- حفظ البيانات محلياً باستخدام localStorage

## البنية التقنية
- React 18 للواجهة
- TailwindCSS للتصميم
- localStorage لحفظ البيانات
- واجهة باللغة العربية مع دعم RTL

## الملفات الرئيسية
- `index.html`: الصفحة الرئيسية
- `app.js`: المكون الرئيسي وإدارة الحالة
- `components/`: مكونات الواجهة
- `utils/storage.js`: وظائف التخزين المحلي
